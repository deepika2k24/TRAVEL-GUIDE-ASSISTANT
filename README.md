# 🧭 Personalized Travel Recommendation System (RAG-based)

## Overview
This project is a **personalized travel planning system** that generates **budget-aware itineraries** using compressed destination guides and retrieval-augmented generation (RAG).  
The system prioritizes **budget constraints first**, ensuring all recommendations stay practical, realistic, and low-latency.

---

## Problem Statement
Most travel planners suggest itineraries first and calculate costs later, often exceeding user budgets.  
This system flips the approach by **optimizing the itinerary around the user’s budget from the start**, while still offering personalized recommendations.

---

## Key Features

### ⭐ Primary Unique Feature
**Budget-First Itinerary Generator**
- Automatically splits the user’s budget (stay, food, travel)
- Generates itineraries strictly within budget limits
- Warns users if constraints cannot be met

### Supporting Features
- **Itinerary Generation**: Day-wise plans based on interests and duration  
- **Local Tips Extraction**: AI-generated local advice and mistakes to avoid  
- **Trip Compression Mode**: Condenses multi-day trips into fewer days  
- **Photo Recommendations**: Highlights top photo-worthy locations  
- **Multi-Destination Planning** (scalable design)
